# AI Based Knowledge Graph Builder for Enterprise Intelligence

## Overview
This project builds an AI-powered Knowledge Graph using Neo4j, 
vector embeddings, and semantic search to analyze enterprise data 
such as customers, tickets, products, and issues.

## Features
- Knowledge Graph construction
- Semantic search using vector embeddings
- Analytical graph queries
- Streamlit dashboard

## Tech Stack
- Python
- Neo4j
- Streamlit
- FAISS
- Sentence Transformers

## Project Structure
backend/ → Graph logic  
frontend/ → Streamlit UI  
data/ → CSV files
